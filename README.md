Harry Potter Hangman Game 🧙‍♂️⚡
A text-based, command-line Hangman game built using Python. Players guess random character names from the Harry Potter universe while tracking their remaining lives through a dynamically rendering visual stick figure.

🚀 Features
1. Theme-Based Word Bank: Randomly selects names from a curated list of Harry Potter characters (e.g., Dumbledore, Hermione, Voldemort).
2. Dynamic Visual Gallows: Renders progressive body parts of the stick figure in the console for every wrong guess.
3. Input Validation: Prevents players from losing lives on repeated guesses by tracking previous entries.
4. Replay System: Prompts the player at the end of a match to easily start a new round without restarting the script.

🛠️ Built With
1. Python 3 (Optimized for Python IDLE and standard terminal environments)
2. random module for automated word selection
